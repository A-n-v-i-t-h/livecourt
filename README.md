# 🏸 LiveCourt

LiveCourt is a real-time badminton court availability dashboard for Hyderabad.

It aggregates live slot data from multiple venues and presents it in a clean, decision-focused interface so players can quickly identify when and where they can play.


🚀 Features

- Real-time court slot availability aggregation
- Multi-venue availability dashboard
- Smart **“Upcoming Slots Only”** filtering for instant decisions
- Dynamic date switching (Today / Tomorrow)
- 12-hour time formatting for user readability
- Court-level slot grouping
- Clean interactive UI with hover highlighting
- Venue price per hour visibility


🧠 Problem Solved

Sports booking apps require users to:

- Open each venue separately  
- Check multiple courts manually  
- Compare time slots inefficiently  

LiveCourt provides a **single unified availability view**, reducing friction in finding playable slots.

---

🛠 Tech Stack

**Frontend**
- React (Vite)
- Inline styled UI components
- Dynamic state-driven rendering

**Backend**
- Node.js
- Express
- Axios
- Real-time API aggregation layer

The backend normalizes venue data, formats time slots, and provides a clean structured response to the frontend.


📍 Current Coverage

- Hyderabad  
- Manikonda badminton venues  

Designed to be easily extendable to additional areas.
